# HTML Live Editor

실시간으로 HTML 파일을 편집할 수 있는 웹 기반 편집기입니다. 드래그 앤 드롭으로 HTML 파일을 업로드하고, 브라우저에서 바로 요소를 편집하거나 텍스트를 수정할 수 있습니다.

**[250922html-editer.netlify.app](https://250922html-editer.netlify.app)** 에서 바로 테스트해보세요!

## 🚀 주요 기능

### 📁 파일 업로드

- **드래그 앤 드롭**: HTML 파일을 브라우저 창에 끌어다 놓기
- **파일 선택**: 버튼을 통해 HTML 파일 선택
- **지원 형식**: .html, .htm 파일

### ✏️ 실시간 편집

- **요소 편집 모드**: HTML 요소 선택, 이동, 복제, 삭제
- **텍스트 편집**: 페이지 내 텍스트 직접 수정
- **시각적 하이라이트**: 편집 가능한 요소에 마우스 오버 시 시각적 표시

### 🔧 편집 기능

- **요소 조작**:

  - 요소 위/아래 이동
  - 요소 복제
  - 요소 삭제
  - 새 요소 추가 (버튼, 링크, 이미지, 리스트 아이템)

- **테이블 전용 기능**:
  - 행/열 추가 (위, 아래, 왼쪽, 오른쪽)
  - 행/열 삭제
  - 테이블 셀 편집

### 🔄 되돌리기/다시실행

- **Ctrl+Z**: 되돌리기 (최대 20단계)
- **Ctrl+Y**: 다시실행
- **히스토리 관리**: 모든 편집 작업 자동 저장

### 💾 파일 저장

- **다운로드**: 편집된 HTML 파일을 새파일로 로컬에 저장

## 🖥️ 사용법

### 1. 파일 업로드

1. 브라우저에서 `index.html` 파일을 실행
2. HTML 파일을 드래그 앤 드롭하거나 "HTML 파일 선택" 버튼 클릭
3. 파일이 전체화면으로 표시됨

### 2. 요소 편집

1. **요소 선택**: 편집하고 싶은 HTML 요소를 클릭
2. **플로팅 툴바**: 선택된 요소 근처에 편집 툴바가 나타남
   - ↑: 요소를 위로 이동
   - ↓: 요소를 아래로 이동
   - 📄: 요소 복제
   - ✖: 요소 삭제

### 3. 텍스트 편집

1. 편집 가능한 텍스트 위에 마우스를 올리면 연한 배경색이 나타남
2. 텍스트를 클릭하여 직접 편집
3. Enter 키를 누르거나 다른 곳을 클릭하여 편집 완료

### 4. 파일 저장

우상단의 "📥 다운로드" 버튼을 클릭하여 편집된 HTML 파일을 저장합니다.
